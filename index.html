Le Workflow de Collaboration

1
Analyse & Annotation (Reviewer)

Chargez le .ifs. Commentez les exigences dans "Checklist" ou "Profil".

2
Envoi du Package (Reviewer)

Générez un .ifsp et transmettez-le à l'auditeur.

3
Réponse & Correction (Auditeur)

L'auditeur répond aux points ouverts dans l'onglet "Tâches".

4
Validation Finale (Reviewer)

Le reviewer valide (Résolu) ou relance jusqu'à clôture complète.

Guide des formats de fichiers

Format Description & Usage Action
.ifs Fichier JSON brut exporté de NEO. Source de l'audit. Import Initial
.ifsp Package d'échange compressé (Reviewer ↔ Auditeur). Transfert
.ifsr Sauvegarde complète de session de travail. Continuité
.xlsx Plan d'actions Excel ou export final des écarts. Rapport / Import PA
